# Jeu du Pendu

## Objectifs
- Implémenter un jeu du pendu en Python
- Utiliser les structures de contrôle
- Manipuler les chaînes de caractères

## Fonctionnalités
- Liste de mots prédéfinis
- Affichage du pendu en ASCII art
- Compteur de tentatives
